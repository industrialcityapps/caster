plotSpace:shouldHandlePointingDeviceCancelledEvent:

@brief (Optional) Notifies that plot space intercepted a device cancelled event.

Parameters

The plot space.

event

The native event (e.g., UIEvent on iPhone)

Return Value

plotSpace:shouldScaleBy:aboutPoint:

@brief (Optional) Informs the receiver that it should uniformly scale (e.g., in response to a pinch on iOS).

- (BOOL)plotSpace:(CPTPlotSpace *)space shouldScaleBy:(CGFloat)interactionScale aboutPoint:(CGPoint)interactionPoint

Parameters

space

The plot space.

interactionScale

The scaling factor.

interactionPoint

The coordinates of the scaling centroid.

Return Value

YES should be returned if the gesture should be handled by the plot space, and NO to prevent handling. In either case, the delegate may choose to take extra actions, or handle the scaling itself.

Discussion

@brief (Optional) Informs the receiver that it should uniformly scale (e.g., in response to a pinch on iOS).
