barFillForBarPlot:recordIndex:

@brief (Optional) Gets a bar fill for the given bar plot.
- (CPTFill *)barFillForBarPlot:(CPTBarPlot *)barPlot recordIndex:(NSUInteger)index

Parameters

barPlot

The bar plot.

index

The data index of interest.

Return Value

The bar fill for the bar with the given index. If the data source returns nil, the default fill is used.

If the data source returns an NSNull object, no fill is drawn.

Discussion

@brief (Optional) Gets a bar fill for the given bar plot.
